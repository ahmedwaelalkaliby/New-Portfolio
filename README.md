# Ahmed Blue Interactive Portfolio

A modern, reusable Next.js portfolio built with TypeScript, Tailwind CSS, Framer Motion, and data-driven sections.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Structure

```txt
app/
components/
  layout/
  sections/
  ui/
lib/
```

## Edit content

Update your profile, experience, skills, and projects in:

```txt
lib/data.ts
```

## Design direction

Minimal Vercel/Linear-inspired dark interface with blue accents, reusable cards, animated reveal patterns, cursor glow, sticky navigation, and responsive project cards.
